# LeadShield Brand Tokens (App + Web)

Use these exact values so LeadShield Android app and website stay visually consistent.

## Logo files

- **Web / marketing:** `branding/web-pack/` (full lockup, symbol, favicons — see `branding/README.md`)
- **Android launcher:** generated into `app/src/main/res/mipmap-*` from `branding/source/mctb-logo-full.png` (updated to LeadShield logo)

## Core Colors

- Primary Blue: `#2563EB`
- Primary Blue Dark: `#1D4ED8`
- Primary Blue Light: `#60A5FA`
- Accent Amber (CTA): `#F59E0B`
- Accent Indigo: `#6366F1`

## Surfaces

- Background: `#111827`
- Elevated Background: `#1F2937`
- Card Background: `#0F172A`
- Border: `#334155`

## Text

- Primary Text: `#FFFFFF`
- Secondary Text: `#E5E7EB`
- Tertiary Text: `#9CA3AF`

## Status

- Success: `#22C55E`
- Warning: `#F59E0B`
- Error: `#EF4444`
- Info: `#3B82F6`

## Usage Rules

- Use blue for primary actions and selected states.
- Use amber for upgrade/paywall CTAs only.
- Keep body text at high contrast on dark surfaces.
- Minimum touch target: 48dp (mobile) / 44px (web).
